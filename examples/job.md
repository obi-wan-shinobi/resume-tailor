# Machine Learning Platform Engineer

**Company:** Picnic  
**Location:** Amsterdam, Netherlands  
**Type:** Full-time  

## About the Role

We are looking for a Machine Learning Platform Engineer to join our ML Platform team. You will be responsible for building and maintaining the infrastructure that enables our data scientists and ML engineers to develop, deploy, and monitor machine learning models at scale.

## Responsibilities

- Design, build, and maintain scalable ML infrastructure and pipelines
- Develop tools and frameworks for model training, evaluation, and deployment
- Build and maintain CI/CD pipelines for ML model lifecycle management
- Implement monitoring and observability for production ML systems
- Collaborate with data scientists to productionize machine learning models
- Optimize model serving for low-latency, high-throughput inference
- Contribute to the development of internal ML platform tools

## Requirements

- 2+ years of experience in software engineering or ML engineering
- Strong proficiency in Python
- Experience with ML frameworks (PyTorch, TensorFlow, or similar)
- Experience with containerization (Docker) and orchestration (Kubernetes)
- Familiarity with CI/CD practices and tools
- Experience with cloud platforms (GCP, AWS, or Azure)
- Understanding of ML model lifecycle (training, evaluation, deployment, monitoring)

## Nice to Have

- Experience with ML pipeline frameworks (Kubeflow, MLflow, Airflow)
- Knowledge of distributed computing and training
- Experience with Go or Rust
- Familiarity with infrastructure-as-code tools (Terraform, Helm)
- Experience with real-time model serving systems
